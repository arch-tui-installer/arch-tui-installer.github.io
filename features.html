This Installer includes:

- Easy to Use OS-Installer

- Easy to Use Desktop-Installer

- Multi Language Support

- UEFI Support

- All installers are graphical

- This project is Opensource

- Xfce4, Cinnamon, GNOME, KDE-Plasma, Deepin, MATE, Trinity, LXQt and LXDE can be installed.

- Three installation modes: Basic, Full and 2Steps (Click to read more)Basic: Installs Archlinux without a desktop.
Full: Installs the basic Archlinux and after a reboot, the desktop will be automaticly installed.
2Steps: Installs Archlinux without any setting and configuratation. Only the partitions will be configured.
After a reboot the user will configure the system with a setup-script (like the OutOfBoxExperience in Windows).

- Automated Partitioning

- Automated Hardware detection for CPU and GPU

- BTRFS-Support

- Integrated Updater so that is Setup is also up-to-date

- Recently updated
